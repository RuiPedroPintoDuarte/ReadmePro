# ReadmePro

A simple, beautiful tool for generating professional README files.

## How to Use

Visit the live app here:

**[https://github.com/RuiPedroPintoDuarte/ReadmePro](https://github.com/RuiPedroPintoDuarte/ReadmePro)**

1. Open the link above in your browser.
2. Fill in your project details using the guided form.
3. Copy or download your generated README.

> No installation required — runs entirely in your browser.
